# Event Board – RTC Driven Message Display System

This project is an embedded system built on the LPC2148 microcontroller.  
It displays scheduled messages using the RTC module, shows temperature from ADC, scrolls event messages on an LCD, and accepts user input through a keypad.

## Features
- RTC-based time & date display  
- Event scheduling (10 predefined messages)  
- LCD message scrolling  
- 4×4 keypad input  
- ADC temperature sensing  
- Password-protected settings  

## Hardware Used
- LPC2148 ARM7 MCU  
- 16x2 LCD  
- 4x4 Keypad  
- LM35 / ADC Temperature Sensor
